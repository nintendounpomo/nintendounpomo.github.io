Introducción

I.A.P. es un utilitario para conducir ataques de perforación en redes IP. Es utilizado por todo tipo de usuarios, desde administradores de sistemas hasta neófitos, para tareas de prevención de ataques y represalias dirigidas. A diferencia de otros sistemas similares, los ataques no van dirigidos al host objetivo, sino contra el usuario mismo (la víctima).

Características

I.A.P. utiliza una variedad de técnicas, desde la creación directa de paquetes IP hasta el aprovechamiento de características de protocolos específicos en capas superiores. Su arquitectura de plugins le permite soportar nuevos protocolos de manera rápida y sencilla, con lo cual es posible definir esquemas de ataque para virtualmente cualquier tipo de objetivo.

  • Funciona en Windows, Linux y OS/X (y próximamente Android).

  • Interfaz de linea de comandos sencilla y flexible.

  • Varios tipos de ataque: rápidos, duros, con vueltas y combinados.

  • Posibilidad de graduar variables que configuran el ataque.

  • Posibilidad de definir tipos de ataque a medida.

Modo de uso

Luego de descargar e instalar el paquete, puede ejecutar el utilitario de la siguiente manera:

# iap [opciones] <ip>[:<puerto>] <objetivo>

Donde ip es la dirección ip del objetivo y objetivo es el nombre del usuario que recibirá el ataque. Opcionalmente se puede especificar el puerto por donde se introducirá la sonda de ataque. Si no se especifica, se utilizará el puerto 79 TCP.

Por ejemplo:

# iap 192.168.0.100 "Chapu"

Conducirá un ataque contra el usuario Chapu conectado en el host con IP 192.168.0.100 dentro de la LAN.

En caso de que el host objetivo esté detrás de un router NAT, se deberá especificar el número de puerto de alguna aplicación que esté ejecutando el usuario para introducirse a través de ella.

Opciones

-s <speed>
  Velocidad del envío de paquetes. 0 es la máxima posible, 256 la más despacio. [Por defecto 128]

-l <love>
  Nivel de amor. 0 sin amor, 128 con todo el amor, -127 para sadismo. [Por defecto 0]

-h <hard>
  Nivel de dureza del ataque. De 0 a 10, 0 es el mínimo, 10 es el máximo. [Por defecto 0]

-t <twist>
  Velocidad de giro de la sonda a medida que penetra en el objetivo. De 0 a 128. [Por defecto 0]

-m <msg>
  Mensaje para mostrar a la víctima luego de una penetración exitosa. [Por defecto "Your anus has been perforated"]
